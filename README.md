
openfire-offlinemessage
==================================

### 插件说明
在用户未出席的情况下实时获取用户离线消息数，并将其推送给客户端

### 安装
* 配置 build.properties中的java.home 和 tomcat.home
* 在Eclipse下build.xml 运行 Run As -> Ant Build
* 在bin/下找到生成的offlinemessage.jar，将其上传到openfire服务端
